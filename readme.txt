La seguente repository è un'api scritta con il linguaggio di programmazzione Go.
Deve essere in esecuzione con il giusto database per poter provare il sito.

Per eseguirlo spostarsi con il terminale nella seguente cartella ed eseguire il seguente comando: "go run ./server.go", naturalmente bisogna aver installato nella macchina Go (scaricabile dal sito ufficiale).

Il file .sql per il database aggiornato si può trovare al seguente link: https://www.dropbox.com/s/6umw55097eccogt/compleo%287%29.sql?dl=0
